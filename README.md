# 917869休闲小游戏
## 概述
一款 4x4 网格的数字合成益智游戏，支持键盘与触控操作，内置规则表驱动的合成逻辑与动态视觉特效。

## 🎮 游戏简介

玩家通过上下左右滑动（或方向键）推动方块移动，按规则让方块发生：

- 合成（合并为新数字）
- 消失（撞击后直接消除）

游戏目标是让一行形成目标序列 `[91, 78, 69]`，达成即胜利。

---

## 🧩 特性

- 4x4 棋盘
- 规则表驱动的合成系统（horizontal / vertical 分方向）
- 消失规则与普通挡板逻辑分离
- 胜负判定
  - 无空位且无可移动时游戏结束
  - 达成横向 `[91, 78, 69]` 为胜利
- 移动输入
  - 键盘方向键 / WASD
  - 触控滑动
  - 长按后可进行上下滚动

## 🚀 运行方式

1. 用浏览器打开 `数字合成游戏动态光效优化.html`
2. 开始滑动或使用方向键操作
3. 重新开始请点击 `新游戏`

> 不需要依赖任何构建工具或服务器，直接本地打开即可。

---

## 🎯 游戏规则

### 生成规则
- 每次在随机空位生成一个随机数字：`6 / 7 / 8`

### 横向合成

7→8=78
13↔78=91
6↔9=69
91↔1=9


### 纵向合成

7→6=1
6→7=13
8→7=1
13↔78=91
91↔1=9

### 消失规则
- 未定义组合但双方都属于 `[1, 6, 7, 8, 9]` 时消失

### 胜利条件
- 任一行出现连续 `91, 78, 69` 时胜利
